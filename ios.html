IOS Navigation App

Previous Next

I used Swift to create an app that recommends nearby murals to users based on their live location. The murals are retrieved via an API, and the user can favourite murals or see more images/details about the murals by tapping on them. I also implemented UserDefaults and Core Data for persistent storage so that a constant internet connection is not required.

React Social Media Platform with Machine Learning Recommendations

Previous Next

These are screenshots of my project. It is a social media platform created using React for the frontend, and Node.js for the backend. It fetches the games from the RAWG API, and displays them to users where they can interact with them by seeing other images, leaving reviews and favouriting them.

These interactions train the machine learning algorithm (made using python) to recommend games.

Users can also search for and add friends, message them and see their reviews and favourite games.

Users passwords are encrypted and stored securely on a SQL server.

A video demo of my project can be seen here. It also provides an insight into the code, database, and the machine learning algorithm.

Technologies Used:

  • React
  • Node.js
  • SQL
  • Python scikit-learn
  • Git
